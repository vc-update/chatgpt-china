# 使用教程

## 快速开始

使用 ChatGPT 中文版非常简单，只需 3 步：

### 1. 访问网站

直接打开 [ChatGPT 中文版](https://gptcat.cc)，无需注册登录。

### 2. 输入问题

在对话框中输入你的问题或需求，例如：
- 帮我写一篇关于春天的作文
- 用 Python 写一个计算器程序
- 翻译这段英文

### 3. 获得回答

点击发送，AI 会立即为你生成回答。

## 使用技巧

### 提问要清晰具体
❌ 不好的提问：写个程序  
✅ 好的提问：用 Python 写一个计算两个数字相加的程序

### 可以多轮对话
如果第一次回答不满意，可以继续追问或要求修改。

### 指定输出格式
可以要求 AI 以特定格式输出，例如：
- 用表格形式总结
- 分点列举
- 用代码块展示

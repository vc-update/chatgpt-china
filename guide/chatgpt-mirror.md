## 1. 什么是 ChatGPT 镜像网站？原理是什么？

很多新手听到“镜像”会觉得不安全，其实正规的镜像站（Mirror Site）是通过技术手段，在国内服务器和 OpenAI 美国服务器之间搭建了一座“桥梁”。

* **技术原理**：你向镜像站提问 -> 镜像站通过企业级 API 转发给 OpenAI -> OpenAI 返回答案 -> 镜像站展示给你。
* **核心优势**：
    1.  **无需魔法**：直接访问国内网页，速度飞快。
    2.  **永不封号**：使用的是服务商的企业账号池，不用你自己注册，彻底规避封号风险。
    3.  **功能增强**：官方没有的 Midjourney 画图、Claude 模型，镜像站通常能聚合在一起。

1.  **问逻辑题**：
    * *问题*：“鲁迅和周树人是同一个人吗？为什么他打了周树人？”

    * 让它写一个贪吃蛇游戏。**GPT-5.2** 能一次性写出可运行的完整代码，而低端模型通常会有 Bug。

## 一、AI 辅助论文写作能做什么？

先明确一点：AI是辅助工具，不是枪手。合理使用AI和学术不端有本质区别。

**AI 可以帮你做的事：**

| 环节 | AI能做什么 | 推荐模型 |
|------|-----------|---------|
| 选题 | 头脑风暴、热点分析、可行性评估 | GPT-5.4 |
| 文献检索 | 提供检索关键词、总结文献要点 | Gemini 3.1 Pro |
| 大纲搭建 | 生成论文结构、章节安排建议 | GPT-5.4 / Claude 4.6 |
| 初稿辅助 | 扩写段落、补充论述、生成过渡句 | GPT-5.4 |
| 数据分析 | 代码编写、统计分析、可视化 | Claude 4.6 |
| 润色修改 | 语法修正、学术语言转换、表达优化 | GPT-5.4 |
| 降重改写 | 换词换句、调整句式、改变表达方式 | GPT-5.4 / Claude 4.6 |
| 翻译 | 中英互译、学术论文翻译 | GPT-5.4 |

**AI 不能替你做的事：**
- ❌ 独立产生原创观点和实验数据
- ❌ 保证引用文献的真实性（AI可能编造文献）
- ❌ 替代你的学术判断和独立思考

## 二、论文写作各环节实操指南

### 环节一：选题阶段

**Prompt模板：**

```
我是[专业]的[本科/硕士/博士]生，研究方向是[方向]。
请帮我：
1. 列出该领域2025-2026年的5个热点研究方向
2. 每个方向给出一个具体的论文选题建议
3. 分析每个选题的创新点和可行性
4. 指出可能的研究难点

要求：选题要有学术价值，且在国内现有条件下可完成。
```

**实测效果**：GPT-5.4在中文学术语境下的选题建议最实用，能准确把握国内学术风向。

### 环节二：文献综述

**Prompt模板：**

```
请以学术论文的文献综述风格，围绕"[主题]"这一研究方向：
1. 梳理该领域的发展历程（3-5个关键阶段）
2. 总结当前主要的研究流派和代表性观点
3. 指出现有研究的不足和未来方向
4. 使用学术写作规范，语言严谨客观

注意：请标注你参考了哪些概念和理论，我会自行查找真实文献。
```

**重要提醒**：AI生成的文献引用经常是编造的！务必自己去知网、Google Scholar等平台核实每一条引用。

Gemini 3.1 Pro的百万上下文窗口在这个环节特别有用——你可以把已经找到的十几篇文献全部上传，让它帮你做交叉对比和综述整理。

### 环节三：论文润色

**Prompt模板：**

```
请帮我润色以下论文段落，要求：
1. 保持学术写作风格
2. 修正语法和表达错误
3. 提升句子的学术性和逻辑连贯性
4. 不改变原意
5. 标注修改了哪些地方及修改原因

[粘贴你的段落]
```

### 环节四：降重改写

**Prompt模板：**

```
请对以下段落进行学术改写，要求：
1. 保持原意不变
2. 更换表达方式和句式结构
3. 替换近义词但保留专业术语
4. 保持学术论文的严谨性
5. 改写幅度要大，不能只是简单换词

[粘贴需要降重的段落]
```

GPT-5.4的中文学术写作能力是目前最强的。它能很好地把握中文论文的行文规范，包括"由此可见""综上所述"等学术过渡用语的自然使用。GPT-5.4 Thinking模式在处理复杂论证时尤为出色。

如果你的论文涉及编程、数据处理或统计分析，Claude 4.6是最佳选择。它的代码能力在四大模型中最强，能帮你写Python数据分析脚本、R语言统计代码等。

Gemini 3.1 Pro的百万上下文窗口让它在处理大量文献时如鱼得水。你可以把整篇论文上传让它检查逻辑一致性，或者同时上传多篇参考文献让它做综述梳理。

## 五、AI写论文避坑指南

### ⚠️ 坑1：虚假引用

AI经常编造不存在的论文和作者。**每一条引用都必须自己核实**，去知网、Google Scholar搜索确认。

### ⚠️ 坑2：过度依赖

AI生成的内容缺乏原创洞见。你的核心论点、创新观点和实验设计必须是自己的。

### ⚠️ 坑3：AI检测

越来越多的学校使用AI检测工具。建议：
- 用AI生成初稿后，大幅度重写和融入自己的思考
- 添加自己的案例、数据和分析
- 调整行文风格，让文章有"人味"

### ⚠️ 坑4：学术规范

AI可能不了解你学校的具体格式要求。引用格式（APA/MLA/GB/T 7714）、页面设置等要自己检查。

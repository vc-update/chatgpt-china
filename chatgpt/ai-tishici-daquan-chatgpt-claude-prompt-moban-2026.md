## 一、写作类Prompt（16个）

### 1.1 公众号爆款文章

```
你是一位10万+公众号的主编，擅长写引发共鸣的爆款文章。
请帮我写一篇关于[主题]的文章：
- 标题：给出5个备选标题，使用数字、悬念或反常识
- 开头：用一个让人"停不下来"的故事或数据开场
- 正文：3-4个核心观点，每个观点配一个生动案例
- 结尾：一句让人想转发的金句
- 字数：2000-3000字
- 风格：像聊天一样轻松，但观点犀利有深度
```

### 1.2 小红书种草文案

```
你是小红书头部博主，擅长写种草笔记。请按以下要求写：
- 产品：[产品名称和简介]
- 标题：带emoji，有数字，引发好奇（给3个备选）
- 正文：先说痛点，再说发现，然后描述体验感受
- 标签：给出10个相关话题标签
- 字数：300-500字
- 风格：真实、口语化、有个人体验感
```

### 1.3 高转化带货文案

```
你是一位资深文案策划师，精通AIDA（注意-兴趣-欲望-行动）文案模型。
请为[产品]写一段带货文案：
- 目标人群：[描述]
- 核心卖点：[1-3个]
- 痛点场景：描述目标用户的困扰
- 解决方案：产品如何解决这个问题
- 行动号召：让用户立即下单的理由
- 字数：200-400字
```

### 1.4 周报/月报自动生成

```
请帮我写本周工作周报，信息如下：
- 本周完成的工作：[列出3-5项]
- 遇到的问题：[描述]
- 下周计划：[列出3-5项]

要求：
1. 用STAR法则（情境-任务-行动-结果）描述每项工作
2. 突出成果和数据（如有）
3. 问题要附带解决方案或思路
4. 语气专业但不生硬
5. 控制在500字以内
```

### 1.5 SEO长文

```
你是一位SEO专家兼内容编辑。请帮我写一篇针对关键词"[关键词]"的SEO文章：
- 标题：包含主关键词，字数50-60字
- 结构：H1标题 + H2分段（至少5个） + H3小节
- 首段：前100字出现主关键词
- 正文：自然嵌入5-8个相关长尾词
- 字数：3000字以上
- 结尾：FAQ部分（至少3个常见问题）
- 风格：专业但易读
```

### 1.6 邮件写作

```
请帮我写一封[商务/求职/感谢/催款]邮件：
- 收件人：[身份/关系]
- 目的：[具体目的]
- 关键信息：[需要传达的要点]
- 语气：[正式/半正式/友好]
- 语言：[中文/英文]
- 字数限制：不超过300字
```

### 1.7 演讲稿/PPT大纲

请给出：
1. 完整的PPT大纲（每页的标题和要点）

3. 3个核心论点及支撑案例

5. 可能的Q&A准备

### 1.8 短视频脚本

## 二、编程类Prompt（12个）

### 2.1 代码生成通用模板

```
你是一位资深[语言]开发工程师。请帮我实现以下功能：

需求描述：[详细描述]

技术要求：
- 语言/框架：[指定]
- 代码规范：[如PEP8/Airbnb]
- 性能要求：[如有]

请提供：
1. 完整可运行的代码
2. 关键逻辑的注释
3. 使用示例
4. 可能的边界情况处理
```

### 2.2 Debug排错

```
以下代码运行时出现[错误描述]：

```[语言]
[粘贴代码]
```

报错信息：
```
[粘贴报错]
```

请：
1. 定位错误的具体位置和原因
2. 解释为什么会出现这个错误
3. 给出修复后的完整代码
4. 说明如何避免类似问题
```

### 2.3 代码审查

```
请对以下代码进行Code Review：

```[语言]
[粘贴代码]
```

请从以下角度评审：
1. 代码质量（可读性、命名规范）
2. 潜在Bug或边界情况
3. 性能优化建议
4. 安全性问题
5. 最佳实践建议
评分：满分10分
```

### 2.4 SQL查询生成

```
数据库表结构如下：
[描述表结构]

请写SQL查询实现：[需求描述]

要求：
- 数据库类型：[MySQL/PostgreSQL/SQLite]
- 考虑性能优化
- 附上解释每一步的注释
```

## 九、Prompt进阶技巧

### 技巧1：角色设定法

在Prompt开头给AI一个具体身份，输出质量提升50%+：
```
❌ "帮我写一篇文章"
✅ "你是一位有15年经验的财经记者，擅长用数据讲故事。帮我写一篇..."
```

### 技巧2：结构化输出

明确告诉AI你要什么格式：
```
❌ "分析一下这个问题"
✅ "请按以下格式分析：1.问题定义 2.原因分析 3.解决方案 4.风险评估"
```

### 技巧3：给出反面示例

告诉AI什么是你不想要的：
```
"请注意：不要使用'赋能''抓手''打通'等商务黑话，用日常用语表达。"
```

### 技巧4：链式思维

让AI一步步思考：
```
"请一步一步分析这个问题，先列出已知条件，再推导逻辑，最后给出结论。"
```

### 技巧5：迭代优化

不满意就追问：
```
"这个版本太正式了，请用更轻松幽默的语气重写，加入2-3个生活化的比喻。"
```

## 常见问题

### Q1：Prompt写得越长越好吗？

不是。关键是**信息密度**。一个200字的精准Prompt比1000字的模糊描述效果好得多。核心是：角色+任务+要求+格式，四要素齐全就够了。

### Q2：中文Prompt还是英文Prompt效果好？

取决于模型和场景。Claude 4.6用中文Prompt效果很好；GPT-5.4在某些技术场景下英文Prompt更精准。建议在 [GPTCat](https://gptcat.cc) 上都试试对比。

### Q3：这些Prompt可以直接复制使用吗？

可以，但建议根据你的实际需求修改方括号内的内容。好的Prompt是"模板+个性化"的结合。
